HTML:----
HTML, or HyperText Markup Language, is a markup language that defines the structure and meaning of web content. It's a set of codes or symbols that are inserted into a file to be displayed on the internet. HTML uses markup to annotate text, images, and other content so that web browsers can display them.
CSS:-----
Cascading Style Sheets (CSS) is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.
